# 🛍️ Mall Customer Segmentation using K-Means Clustering

## 📌 Project Overview
This project focuses on segmenting mall customers into distinct groups based on their purchasing behavior using unsupervised machine learning techniques.

Customer segmentation helps businesses understand different customer groups and target them effectively.

---

## 📊 Dataset
- Dataset: `Mall_Customers.csv`
- Features include:
  - Customer ID
  - Gender
  - Age
  - Annual Income
  - Spending Score

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

---

## 🔍 Project Workflow

1. **Data Preprocessing**
   - Handling missing values
   - Feature selection

2. **Exploratory Data Analysis (EDA)**
   - Distribution of age, income, spending score
   - Correlation analysis

3. **Clustering**
   - Applied K-Means clustering
   - Used Elbow Method to find optimal number of clusters

4. **Visualization**
   - Visualized customer segments using scatter plots

---

## 📈 Results
- Identified distinct customer segments based on income and spending
- Useful for targeted marketing and business strategy
